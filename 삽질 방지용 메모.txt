1. 첫번째 삽질
깃에서 파일을 받은 후 보면 layout에 xml파일이 rendering problem하면서 지랄할때가 있습니다.
지웠다 다시 해보고 병을 해보았지만 원인은 프로젝트 버전이 맞지 않아서였습니다.
간단한 해결 방법은 file->invalidate cashes/restart를 하는 겁니다.
그러면 대부분 된다고 하는데 저는 또 안되더군요ㅎ
그래도 안되면 apk버전을 변경해봅니다.(5.1버전으로 개발을 하므로 최소 api level 22까지 있겠지용)
그래도 안된다…. 그러면 폰 설정 바꿔보고 테마 설정 바꿔보면 됩니다.
저는 테마 설정 하나씩 바꿔보니까 드디어 되더군요ㅎ

2. 두번째 삽질
에뮬레이터 실행할때 오류입니다.  
Android Studio Build Error : This version of Android Studio is incompatible with the Gradle Plugin used….
이런거 나오면 거기 설명대로 instant run 설정 끄면 됩니다.

File->Settings(단축키 컨트롤 알트 에스)->Build, Execution, Deployment -> Instant Run
체크된거 해제!

여기서도 안된다면 저는 해결되서 구글에 검색해 보세요.
